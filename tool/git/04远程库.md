## 远程库相关命令
- 创建远程库别名
    ``` sh
    git remote add 远程库别名 远程库链接
    ```
- 查看远程库
    ``` sh
    git remote -v
    ```
- 推送
    ``` sh
    # 注意：push远程库代码时，本地库版本必须要高于远程库版本，如果本地库代码版本过低，一定要pull先拉取远程库的代码
    git push 远程库别名 分支名
    ```
- 拉取
    ``` sh
    git pull 远程库名 分支名
    ```
- 克隆
    ``` sh
    # 注意，clone会自己做以下操作：
        # 拉取代码
        # 初始化本地库
        # 创建别名(clone创建的别名叫做origin)
    git clone 远程库链接
    ```

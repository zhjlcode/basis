## 开发环境的搭建
环境：
 - Ubuntu 22.4桌面版本
 - VMware 16版本

中文切换：window键 + 空格
打开终端：ctrl+alt+t

## 编译器，调试器安装
- 安装GCC,GDB
    ``` sh
    sudo apt update

    sudo apt install buile-essential gdb
    ```

- 确认安装完成
    ``` sh
    gcc --version
    g++ --version
    gdb --version
    ```

## CMake安装
- 安装CMake
    ``` sh
    sudo apt install cmake
    ```

- 确认安装完成
    ``` sh
    cmake --version
    ```
